 Hospital Management System

This is a Hospital Management System designed to simplify hospital operations by allowing administrators, doctors, and patients to manage healthcare services through a secure web-based platform. 
The system provides features such as user authentication, appointment scheduling, patient management, doctor management, and prescription handling with an intuitive and responsive interface.

 Features

-  User Authentication (Login & Registration)
-  Doctor Management
-  Patient Management
-  Appointment Booking System
-  Prescription Management
-  Appointment History
-  Admin Dashboard
-  Doctor Dashboard
-  Patient Dashboard
- Role-Based Access Control
-  Responsive Design
-  More features coming soon...

---

## Project Overview

The Hospital Management System is a web-based application developed to streamline hospital management and improve communication between administrators, doctors, and patients. It enables efficient appointment scheduling, patient record management, and healthcare service administration while following the MVC (Model-View-Controller) architecture for better maintainability and scalability.

---

##  Technologies Used

- PHP – Backend Development
- MySQL – Database Management
- HTML5 – Web Structure
- CSS3 – Styling and Responsive Design
- JavaScript – Client-side Functionality

- MVC Architecture – Project Structure

 User Roles

 Admin
- Manage doctors
- Manage patients
- Manage appointments
- Monitor system activities
- Admin can access patient records and prescriptions
- Admin can access payment history and status tracking

 Doctor
- View appointment requests
- Manage schedules
- Create prescriptions
- Access patient information
- Doctor can access patient records and prescriptions

 Patient
- Register/Login
- Book appointments
- View appointment history
- View prescriptions
- Update profile
